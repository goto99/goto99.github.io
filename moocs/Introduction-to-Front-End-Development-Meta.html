Alignment basics

HTML Element Center Alignment

To align a child div element to the center, set its margin property to "auto". Notiche that this only works for block-level element, for inline element, you need to change it to a block-level element.

HTML Element Left/Right Alignment

The two most common ways to left and right align elements are to use the float property and the position property.
